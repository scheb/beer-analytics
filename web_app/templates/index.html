Welcome to Beer Analytics

The analytical beer recipe database

Beer Analytics is a database of beer brewing recipes, built specifically for data analysis. It is made for beer enthusiasts and (home)brewers to provide detailed insights how brewing recipes are composed, even when you're not an expert in data analysis. Currently, the database contains {{ recipes }} recipes, mostly from the homebrewing community.

Start exploring how certain beer styles are typically composed or browse the index of fermentables, malts, hops and yeasts to learn how these ingredients are used.

Looking for something even more specific? Use the filter options in the sidebar in the top menu to specify exactly what you're looking for and get customized data analysis results.
